# VibeKeypad Presets

Configuration files and printable icons for VibeKeypad.

## Claude Code

- **[claude_code.sayobak](./claude_code.sayobak)** — SayoDevice configuration file
- **[icons-claude-code.html](./icons-claude-code.html)** — Printable icons (10×12mm)

| Key | Icon | Action | Shortcut |
|-----|------|--------|----------|
| 1 | 🤙🏻 | Yes / Accept | Enter |
| 2 | ⏭ | Yes, don't ask again | 2 |
| 3 | 🙅‍♂️ | No / Reject | 3 + ESC |
| 4 | ❌ | Stop | Ctrl + ESC + C |
| 5 | 👀 | Full output | Ctrl + O |
| 6 | 🎤 | Dictate | Cmd + D |

## How to Import

1. Open [sayodevice.com](https://sayodevice.com) in Chrome
2. Connect your keypad via USB
3. Click **Import** and select the `.sayobak` file
4. Click **Save** to write to device

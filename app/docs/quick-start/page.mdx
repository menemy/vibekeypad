# Quick Start Guide

Get your VibeKeypad up and running in minutes. This guide covers the essential steps to configure and use your keypad.

## Setup Process

### 1. Open the Configuration Tool

Navigate to [https://sayodevice.com/](https://sayodevice.com/) in **Google Chrome**.

> **Important:** Chrome is required because it supports WebHID API for device configuration.

### 2. Connect Your Device

Connect your VibeKeypad using one of these methods:

- **USB Cable** - Direct connection via USB-C
- **USB Dongle** - Wireless receiver (if included with your device)

When prompted by Chrome, select your device and click **Connect**.

### 3. Update Firmware (Optional but Recommended)

1. Check your current firmware version in the SayoDevice interface
2. If an update is available, click **Update Firmware**
3. Wait for the update to complete (do not disconnect during this process)
4. The device will restart automatically

### 4. Configure Your Key Combinations

VibeKeypad supports **6 different key combinations** by default.

#### Basic Configuration (6 Keys)

Map each key to your desired action:

| Key | Suggested Mapping | Action |
|-----|------------------|---------|
| **Key 1** | `Tab` | Accept AI suggestion |
| **Key 2** | `Alt + ]` | Next suggestion |
| **Key 3** | `Cmd/Ctrl + Shift + A` | Ask AI |
| **Key 4** | `Alt + R` | Redo/Regenerate |
| **Key 5** | `Ctrl + R` | Show verbose output |
| **Key 6** | `Fn + Fn` | Push-to-talk voice |

#### Advanced: Using Layers (10 Combinations)

Enable layers to unlock **10 different combinations** using just 6 physical keys:

1. In SayoDevice, navigate to **Layers** settings
2. Assign **one key as a layer modifier** (e.g., Key 6)
3. Configure alternative actions for keys 1-5 when the modifier is held

This gives you 10 unique actions with 6 keys!

### 5. Print and Install Icons

1. Download the icon sheet from the [GitHub repository](https://github.com/menemy/vibekeypad)
2. Print on waterproof vinyl sticker paper (12×12mm per icon)
3. Cut the icons to size
4. Remove keycaps from your switches
5. Apply icons to the top of each keycap
6. Reinstall keycaps

> **Pro Tip:** Use a Cricut or Silhouette cutting machine for precise cuts, or carefully cut by hand with a hobby knife and ruler.

### 6. Start Using Your Keypad

After configuration, your VibeKeypad **works as a standard keyboard**:

- No additional software required
- Works with any application
- Plug and play across different computers
- Your settings are stored in the device

## Example Workflow

Here's how a typical AI coding session works with VibeKeypad:

1. **Claude suggests code** → Press Key 1 to accept
2. **Want to see alternatives?** → Press Key 2 for next suggestion
3. **Need clarification?** → Press Key 3 to open chat
4. **Not quite right?** → Press Key 4 to regenerate
5. **Need more detail?** → Press Key 5 for verbose mode
6. **Voice input faster?** → Hold Key 6 and speak

All without touching your mouse or main keyboard!

## Troubleshooting

### Device Not Detected

- Try a different USB port
- Use a data-capable USB cable (not just charging cables)
- Restart Chrome and reconnect
- Grant USB permissions when Chrome prompts

### Keys Not Responding

- Make sure you saved your configuration in SayoDevice
- Disconnect and reconnect the device
- Check that the application you're using accepts the key combinations
- Test in a text editor to verify keys are sending correctly

### Firmware Update Failed

- Do not disconnect during update
- Try a different USB cable
- Connect directly to computer (not through a hub)
- Download and manually flash firmware if web update fails

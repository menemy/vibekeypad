# Quick Start Guide

Get your VibeKeypad up and running in minutes. This guide covers the essential steps to configure and use your keypad.

## Setup Process

### 1. Open the Configuration Tool

Navigate to [https://sayodevice.com/](https://sayodevice.com/) in **Google Chrome**.

> **Important:** Chrome is required because it supports WebHID API for device configuration.

### 2. Connect Your Device

Connect your VibeKeypad using one of these methods:

- **USB Cable** - Direct connection via USB-C
- **USB Dongle** - Wireless receiver (if included with your device)

When prompted by Chrome, select your device and click **Connect**.

### 3. Update Firmware (Optional but Recommended)

1. Check your current firmware version in the SayoDevice interface
2. If an update is available, click **Update Firmware**
3. Wait for the update to complete (do not disconnect during this process)
4. The device will restart automatically

### 4. Configure Your Key Combinations

VibeKeypad supports **6 keys** that you can map to any shortcuts. See our preset guides for recommended configurations:

- [Claude Code Preset](./claude-code-preset.html)
- [Cursor Agent Preset](./cursor-agent-preset.html)

#### Claude Code Example

| Key | Icon | Shortcut | Action |
|-----|------|----------|--------|
| **Key 1** | 🤙🏻 | `Enter` | Yes / Accept |
| **Key 2** | ⏭ | `2` | Yes, don't ask again |
| **Key 3** | 🙅‍♂️ | `3 + ESC` | No / Reject |
| **Key 4** | ❌ | `Ctrl + C` | Stop |
| **Key 5** | 👀 | `Ctrl + O` | Full output |
| **Key 6** | 🎤 | `Cmd + D` | Dictate |

#### Cursor Agent Example

| Key | Icon | Shortcut | Action |
|-----|------|----------|--------|
| **Key 1** | ✅ | `Enter` | Apply |
| **Key 2** | ⏭ | `Shift + Enter` | Apply + Allowlist |
| **Key 3** | 🙅‍♂️ | `ESC` | Skip |
| **Key 4** | ❌ | `Ctrl + C` | Cancel |
| **Key 5** | 💭 | `Win + .` | Mode toggle |
| **Key 6** | 🎤 | `Shift + Win + Space` | Dictate |

### 5. Print and Install Icons

1. Print icons on regular paper (12×12mm per icon)
2. Cut the icons to size
3. Open transparent keycaps and slide icons inside
4. Snap keycaps back together
5. Press onto switches

> **Tip:** You can use emoji from the tables above or create your own icons.

### 6. Start Using Your Keypad

After configuration, your VibeKeypad **works as a standard keyboard**:

- No additional software required
- Works with any application
- Plug and play across different computers
- Your settings are stored in the device

## Example Workflow

Here's how a typical AI coding session works with VibeKeypad:

1. **Claude suggests a change** → Press 🤙🏻 to accept
2. **Multiple similar changes?** → Press ⏭ to accept all
3. **Wrong suggestion?** → Press 🙅‍♂️ to reject
4. **Going wrong direction?** → Press ❌ to stop
5. **Need details?** → Press 👀 for full output
6. **Complex explanation?** → Press 🎤 and speak

All without touching your mouse or main keyboard!
